# ヨリプラ AIプログラム プロンプト集

就活・仕事・生活で使えるAIプロンプト例をまとめたページです。GeminiなどのAIチャットにコピペして使えます。

## 公開ページ（GitHub Pages）

リポジトリをGitHubにpushしたあと、次の手順でGitHub Pagesを有効にすると公開できます。

1. リポジトリの **Settings** → **Pages**
2. **Source**: Deploy from a branch
3. **Branch**: `main`（または `master`） / フォルダ: **/ (root)**
4. **Save** をクリック

数分後、`https://<あなたのユーザー名>.github.io/yoripla-ai-prompts/` で表示されます。

## ローカルで確認

`index.html` をブラウザで開くか、簡易サーバーで表示できます。

```bash
# 例: Python 3
python -m http.server 8000
# http://localhost:8000 で開く
```

## 構成

- **就活編**: 自己PR、面接練習、志望動機、企業応募メール
- **仕事編**: メール下書き、報告書の構成、議事録のポイント・要約
- **生活編**: レシピ、悩み整理、趣味、通院の体調説明、体調管理

各プロンプトは「コピー」ボタンでクリップボードにコピーできます。

## 注意

- 個人情報をプロンプトに入力しないでください。
- 服薬・医療の判断は医師に相談してください。
- AIの回答は誤りを含む場合があります。考える材料として使い、大事なことは専門家や周りの人と相談してください。
